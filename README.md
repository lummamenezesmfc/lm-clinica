# LM Clínica

Sistema de gestão para clínica estética.

## Funcionalidades previstas

- **Agenda de pacientes** — agendamento com lembretes automáticos
- **Prontuário eletrônico** — histórico, evoluções e fotos antes/depois
- **Controle financeiro** — recebimentos, comissões e fluxo de caixa
- **Gestão de estoque** — controle por validade e movimentação
- **Documentos digitais** — termos e contratos com assinatura digital
